# Basic Sign Up View

A simple Android Sign Up screen created using **XML Layouts** and **Kotlin** in Android Studio. This project demonstrates basic UI design concepts such as image integration, text views, input fields, and responsive layout arrangement.

## 📱 Preview

This project contains a basic sign-up user interface with:

- App logo/image
- Welcome text
- User input fields
- Sign Up button
- Clean and simple XML-based design
- Beginner-friendly Android UI implementation

## 🚀 Features

- Built using Android Studio
- XML Layout Design
- Kotlin Programming Language
- Responsive UI Components
- ImageView and TextView implementation
- Material Design-inspired layout structure

## 🛠️ Technologies Used

- Android Studio
- Kotlin
- XML Layouts
- ConstraintLayout / RelativeLayout
- Android SDK
